    介 绍

    Issues 助手是一个轻松帮你自动管理 issues 的 GitHub Action。

    GitHub Actions 是什么？

    GitHub Actions 是由 GitHub 官方提供在存储库中自动化、自定义和执行软件开发工作流程。您可以发现，创建和共享操作以执行所需的任何工作（包括CI / CD），并在完全定制的工作流程中组合操作。更多介绍。

    issues-helper 就是以此为基础，利用 GitHub Actions 来帮你处理各种关于 issue 方面的操作。

    ✨ 特性

    • 😎 完全免费
    • 🚀 全自动操作
    • 🏖 托管于 GitHub 服务器，只要 GitHub 不宕机，它就不受影响

    ⚡ 反馈

    非常欢迎你来尝试使用，并提出意见，你可以通过以下方式：

    • 通过 Issue 报告 bug 或进行咨询
    • 通过 Discussions 进行讨论
    • 提交 Pull Request 改进 issues-helper 的代码
